# 🖋️ Ink Consórcio

Sistema web desenvolvido para gerenciamento de consórcios de tatuagem, permitindo o controle de clientes, planos, parcelas, pagamentos e contemplações de forma simples, organizada e segura.

---

## 📋 Sobre o Projeto

O **Ink Consórcio** foi criado para facilitar a administração de estúdios de tatuagem que trabalham com planos de consórcio. O sistema oferece um painel administrativo completo e uma área exclusiva para clientes acompanharem seus pagamentos e parcelas.

---

## ✨ Funcionalidades

### Área do Cliente

* Login do cliente
* Visualização dos planos contratados
* Consulta das parcelas
* Acompanhamento do status dos pagamentos
* Pagamento das mensalidades
* Histórico de pagamentos

### Área Administrativa

* Login administrativo
* Cadastro de clientes
* Gerenciamento de planos
* Controle de parcelas
* Gerenciamento de pagamentos
* Atualização do status das parcelas
* Acompanhamento dos participantes

### Integrações

* Mercado Pago
* Simulação de pagamentos para testes
* Retorno automático de pagamentos

---

## 🛠️ Tecnologias Utilizadas

* PHP
* MySQL
* HTML5
* CSS3
* JavaScript
* Mercado Pago API

---

## 💳 Pagamentos

O sistema possui integração com o Mercado Pago para geração de pagamentos das parcelas.

Também possui um modo de simulação para facilitar os testes durante o desenvolvimento.
